Signature POSIX_SIGNAL (basis)

Symbolic names of POSIX signals.

The structure Posix.Signal defines the symbolic names of all the POSIX signals (see Section 3.3 of the POSIX standard 1003.1,1996), and provides conversion functions between them and their underlying representations.

[eqtype signal]
The type of a POSIX signal, an asynchronous notification of an event.

[toWord signal]
[fromWord w]
These funtions convert between a signal identifier and its underlying integer representation. Note that fromWord does not check that the result corresponds to a valid POSIX signal.

Discussion
The name of the corresponding POSIX signal can be derived by capitalizing all letters and adding the string ``SIG'' as a prefix. For example, the POSIX signal associated with usr2 is SIGUSR2.
